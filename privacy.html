FinMaster is a personal finance management application designed to help users track income, expenses, budgets, and financial summaries. Your privacy and data security are extremely important to us.

This Privacy Policy explains how FinMaster handles your data.

1. Information We Collect

FinMaster does not collect personal data on any server.

All financial data you enter (such as income, expenses, budgets, and accounts) is:

  • Stored locally in your browser, and/or
  • Backed up only if you explicitly choose to, using your own Google Drive.

FinMaster does not collect:

  • Names
  • Email addresses
  • Google account details
  • Financial data on any external server

2. How Your Data Is Stored

FinMaster uses an offline-first architecture.

Data storage methods:

  • Local Storage / IndexedDB in your browser
  • Optional encrypted backup to your Google Drive (App Data folder)

When using backup:

  • All data is encrypted on your device
  • Encryption uses a password that only you know
  • The password is never stored
  • If the password is lost, the backup cannot be recovered

3. Google Drive Integration

FinMaster integrates with Google Drive only when you choose to enable backup.

Details:

  • Scope used: https://www.googleapis.com/auth/drive.appdata
  • Access is limited to the app's private App Data folder
  • FinMaster cannot view, access, or modify other files in your Drive
  • No Google access tokens are stored permanently

Google authentication tokens are:

  • Stored in session storage only
  • Automatically cleared when the browser tab is closed

4. Data Security

We take security seriously:

  • Client-side encryption using AES-GCM (256-bit)
  • Key derivation using PBKDF2
  • No server-side storage
  • No background data transmission
  • No analytics or tracking

Your data remains under your full control at all times.

5. User Control

You have full control over your data:

  • You can delete your local data at any time
  • You can create or delete backups manually
  • You can disconnect Google Drive access at any time from your Google Account settings

6. Data Sharing

FinMaster does not share data with any third parties.

There are:

  • No ads
  • No trackers
  • No third-party analytics services

7. Changes to This Policy

This Privacy Policy may be updated in the future to reflect improvements or changes in functionality.

Any updates will be posted on this page.

8. Contact

If you have questions or concerns about this Privacy Policy, you may contact:

Email: linzhijun87@gmail.com

FinMaster adalah aplikasi pengelolaan keuangan pribadi yang membantu pengguna mencatat pendapatan, pengeluaran, anggaran, dan ringkasan keuangan. Privasi dan keamanan data Anda adalah prioritas utama kami.

Dokumen ini menjelaskan bagaimana FinMaster menangani data Anda.

1. Data yang Kami Kumpulkan

FinMaster tidak mengumpulkan data pribadi ke server mana pun.

Semua data keuangan yang Anda masukkan:

  • Disimpan secara lokal di browser Anda, dan/atau
  • Dicadangkan hanya jika Anda memilih sendiri, ke Google Drive milik Anda

FinMaster tidak mengumpulkan:

  • Nama
  • Alamat email
  • Informasi akun Google
  • Data keuangan ke server eksternal

2. Cara Penyimpanan Data

FinMaster menggunakan pendekatan offline-first.

Metode penyimpanan:

  • Local Storage / IndexedDB di browser
  • Backup terenkripsi opsional ke Google Drive (App Data folder)

Saat menggunakan backup:

  • Data dienkripsi di perangkat Anda
  • Menggunakan password yang hanya Anda ketahui
  • Password tidak disimpan
  • Jika password hilang, data backup tidak dapat dipulihkan

3. Integrasi Google Drive

Integrasi Google Drive hanya aktif jika Anda mengizinkannya.

Detail:

  • Scope: https://www.googleapis.com/auth/drive.appdata
  • Akses terbatas hanya ke folder data aplikasi
  • FinMaster tidak dapat mengakses file lain di Google Drive Anda
  • Token Google tidak disimpan secara permanen

Token autentikasi:

  • Disimpan sementara di session browser
  • Akan hilang saat tab ditutup

4. Keamanan Data

Langkah keamanan yang digunakan:

  • Enkripsi sisi klien (AES-GCM 256-bit)
  • PBKDF2 untuk derivasi kunci
  • Tidak ada server backend
  • Tidak ada pengiriman data otomatis
  • Tidak ada pelacakan atau analitik

Data sepenuhnya berada dalam kendali Anda.

5. Kendali Pengguna

Anda dapat:

  • Menghapus data kapan saja
  • Membuat atau menghapus backup secara manual
  • Mencabut akses Google Drive kapan saja

6. Pembagian Data

FinMaster tidak membagikan data kepada pihak ketiga.

Tidak ada:

  • Iklan
  • Tracker
  • Layanan analitik pihak ketiga

7. Perubahan Kebijakan

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu.

Perubahan akan ditampilkan di halaman ini.
